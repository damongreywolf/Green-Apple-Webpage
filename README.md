# Green Apple Gourmet Sandwiches Website

This is a static no-build version of the Green Apple Gourmet Sandwiches website.

## Vercel setup

This version does not need npm, React, Vite, or a build command.

On Vercel, use:
- Framework Preset: Other
- Build Command: leave blank
- Output Directory: leave blank or use .
- Install Command: leave blank

The only required file is index.html.
