# Green Apple Food Truck Website

This is the website for Green Apple Gourmet Sandwiches, built with React and Vite.

## Vercel settings

Framework Preset: Vite  
Build Command: npm run build  
Output Directory: dist  
Install Command: npm install

## Main files

- `src/App.jsx` contains the webpage.
- `src/main.jsx` starts the React app.
- `index.html` loads the app and Tailwind CDN.
